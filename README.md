# SpringUserSystem

## Prerequisites
- JDK 1.7 or later
- Maven 3 or later

## Stack
- Spring Security
- Spring Boot
- Spring Data JPA
- Maven
- JSP
- MYSQL

## Run
```create a mySQL database called "piexchange"```  
```the database should be at localhost:3306/piexchange```  
```mvn clean spring-boot:run```  
```go to http://localhost:8080/```
